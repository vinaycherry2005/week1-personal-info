# Personal Information Manager

## Project Description
This is a beginner Python program that collects and displays personal information using user input.

## What I Learned
- Variables in Python
- Taking user input
- Using loops for validation
- Formatting output using f-strings

## How to Run the Program
1. Install Python
2. Open terminal
3. Navigate to the project folder
4. Run: python personal_info.py

## Features
- Stores personal details
- Validates empty input
- Calculates age in months
- Displays formatted output
